# javascript-Array

This project demonstrates:

- Simple Arrays (Numbers, Strings)
- Mixed Type Arrays
- Multidimensional Arrays (2D Arrays)
- Arrays of Objects
- Array Methods (push, pop, shift, unshift, slice, splice, join, indexOf, includes)
- Looping using `forEach()` and `map()`
